# Vivado Xilinx Projects — Digital Design & RTL Implementation

This repository contains a curated collection of digital logic designs and RTL
implementations developed using **Xilinx Vivado Design Suite**. Each project is
built from the ground up using **Verilog HDL**, covering a wide spectrum of
digital design concepts — from fundamental combinational circuits to structured
sequential logic systems.

The designs in this repository are inspired by industry-standard practice
problems sourced from **HDLBits**, a widely recognized platform for Verilog
skill-building, and are implemented with a strong emphasis on correctness,
readability, and real-world applicability. Every module is accompanied by a
dedicated **testbench** that rigorously validates functionality across all
possible input combinations, ensuring simulation results match expected
theoretical outputs.

Each project follows a consistent and professional workflow:
- **RTL Design** — clean, well-commented Verilog source files
- **Behavioral Simulation** — functional verification using Vivado's built-in
  simulator with waveform analysis
- **RTL Schematic View** — gate-level visualization of the synthesized design
- **Netlist Inspection** — post-synthesis netlist review for deeper hardware
  understanding

The repository is organized into clearly defined categories — starting with
**Combinational Logic**, which includes implementations such as basic logic
gates, multiplexers, byte reversal circuits, vector operations, chip
replications, and arithmetic multipliers. Future additions will progressively
cover **Sequential Logic** (flip-flops, registers, counters), **Finite State
Machines (FSMs)**, and **FPGA-targeted designs**.

This repository serves as a practical, hands-on portfolio demonstrating
proficiency in:
- **Verilog HDL** coding and simulation
- **Xilinx Vivado** design and verification flow
- **Digital logic design** principles and best practices
- **Testbench development** and functional verification
- **RTL-to-schematic** interpretation and netlist analysis

Whether you are a fellow learner, a collaborator, or a recruiter evaluating
RTL design skills, this repository reflects a structured and growing journey
through the world of **digital hardware design and FPGA development**.
